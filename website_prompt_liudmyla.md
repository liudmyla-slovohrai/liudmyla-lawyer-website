# Промт для розробника / дизайнера сайту
## Персональний сайт юриста — ЛЮДМИЛА

---

## 1. Загальна концепція

Персональний сайт-лендінг для юриста зі спеціалізацією на управляючих компаніях та забудовниках. Сайт має передавати три речі з першої секунди: **довіра**, **компетентність**, **конкретність**.

Структура — одна довга сторінка (single-page landing) з навігацією по якорях. Мобільна версія обов'язкова.

---

## 2. Дизайн-стиль: Legal Minimalism with Gold Accent

### Загальний характер
Стиль поєднує два шари:
- **Темний hero** — перший екран з темним або глибоким фоном, великою типографікою та фото. Відчуття авторитетності та ваги.
- **Чистий білий контент** — усі наступні блоки на білому або дуже світлому тлі. Повітря, читабельність, порядок.

Немає яскравих кольорів. Немає дитячих градієнтів. Немає перевантаженості. Все — стримано та впевнено.

### Інспірація
Два референси задають напрямок:

**Референс 1 (ADVMARC):** темний hero з великим заголовком та золотою акцентуацією, статуя Феміди як візуальний символ, цифрові досягнення у колах, команда юристів з фото.

**Референс 2 (Minimalist Lawyer):** ultra-clean білий фон, великі serif-заголовки, нумерований список практик як головний елемент, фото юриста в інтер'єрі кабінету, відгуки клієнтів.

**Для цього сайту:** беремо темний hero з першого референсу + мінімалізм та типографіку з другого.

## 3. Кольорова палітра

| Роль | Колір | HEX |
|------|-------|-----|
| Основний фон (hero) | Глибокий темно-сірий / майже чорний | `#1a1a1a` або `#0f0e0d` |
| Основний фон (контент) | Чистий білий | `#ffffff` |
| Другорядний фон (секції) | Дуже світлий теплий кремовий | `#f8f6f2` |
| Акцентний (золото) | Приглушене тепле золото | `#c9a84c` або `#b8973a` |
| Текст основний | Майже чорний | `#1a1a1a` |
| Текст другорядний | Середньо-сірий | `#6b6b6b` |
| Текст на темному фоні | Білий / кремово-білий | `#f5f2ec` |
| Рамки та лінії | Дуже світлий сірий | `#e0ddd8` |

**Правило акценту:** золото використовується точково — для одного-двох слів у заголовку, кнопок, іконок, горизонтальних розділювачів, цифр у статистиці. Не більше 10–15% від усіх кольорів на сторінці.

## 4. Типографіка

### Пара шрифтів
| Шрифт | Використання | Тип |
|-------|-------------|-----|
| **Cormorant Garamond** або **Playfair Display** | Заголовки H1, H2, великі цитати | Serif, Google Fonts |
| **Inter** або **DM Sans** | Тіло тексту, підписи, навігація, кнопки | Sans-serif, Google Fonts |

Ця пара дає юридичну серйозність (serif) + сучасну читабельність (sans-serif).

### Розміри та ваги
```
H1 (hero)          — 56–72px,  Cormorant Garamond, weight 600–700
H2 (секції)        — 36–44px,  Cormorant Garamond, weight 600
H3 (підзаголовки)  — 20–24px,  Inter, weight 600
Body               — 16–18px,  Inter, weight 400, line-height 1.7
Caption / note     — 13–14px,  Inter, weight 400, color #6b6b6b
Navigation         — 14px,     Inter, weight 500, letter-spacing 0.05em
Кнопки             — 13–14px,  Inter, weight 600, UPPERCASE, letter-spacing 0.1em
```

### Золотий акцент у тексті
У заголовках одне-два ключових слова виділяються золотим кольором:
```
Юридична            ← чорний
допомога для        ← чорний
управляючих         ← чорний
<span style="color: #c9a84c">компаній</span>  ← золотий
```

### БЛОК 1 — Hero (перший екран)

**Висота:** 100vh (повна висота екрану)
**Фон:** темний — або суцільний `#1a1a1a`, або напівпрозорий overlay поверх фото (судовий зал, бібліотека, кабінет). Overlay: `rgba(15, 14, 13, 0.78)`.
**Фото юриста:** справа, на весь висоту блоку, без фону (обрізане або на overlay). Легкий parallax-ефект при скролі — бажано.

**Навігація (header):**
- Прозора поверх hero, фіксована при скролі (стає темною/білою)
- Зліва: ім'я "ЛЮДМИЛА" або логотип
- Справа: Home | Про мене | Послуги | Як я працюю | FAQ | Контакти
- Остання кнопка — золота з рамкою: "Зв'язатися"

**Контент hero (зліва або по центру):**
```
[мала золота лінія або тег — 1px горизонтальна риска + текст]
Юрист • Київ • 15+ років досвіду

[H1 великий serif]
Правовий захист
для управляючих
компаній та
забудовників ←(золотий)

[підзаголовок Inter, кремовий, 18px]
Господарське, житлове та договірне право.
Від консультації до захисту у суді.

[дві кнопки]

Кнопка 2: прозора, рамка #c9a84c, текст кремовий
```

**Фон:** темний (продовження hero) або перехід до кремового
**Макет:** 3 колонки по центру

```
┌──────────────┐  ┌──────────────┐  ┌──────────────┐
│     15+      │  │  [_цифра_]   │  │  [_цифра_]   │
│    Років     │  │  Клієнтів    │  │   Справ      │
```

Цифри — великий serif золотий. Підписи — маленький sans-serif сірий.

> **Примітка розробнику:** значення у квадратних дужках — плейсхолдери. Клієнт заповнить пізніше.

## 6. UI-деталі та мікроанімації

### Кнопки
```
Первинна кнопка:
  background: #c9a84c
  color: #1a1a1a
  padding: 14px 32px
  border-radius: 2px (майже квадратна — юридична стриманість)
  font: Inter 600, 13px, UPPERCASE, letter-spacing 0.1em
  hover: background darken 10%, transform: translateY(-1px)

Вторинна кнопка (ghost):
  background: transparent
  border: 1px solid #c9a84c
  color: #c9a84c
  (решта — як первинна)
  hover: background: rgba(201,168,76,0.1)
```

### Навігація при скролі
- При скролі вниз: header стає `background: rgba(26,26,26,0.96)`, backdrop-filter: blur(8px)
- Активна секція в навігації: підкреслення золотою лінією 2px

### Горизонтальні розділювачі
```
<hr style="border: none; border-top: 1px solid #e0ddd8; margin: 48px 0;">
Або золота коротка лінія для акценту:
<div style="width: 48px; height: 2px; background: #c9a84c; margin: 16px 0 24px;"></div>
```

### Scroll-анімації
Кожен блок з'являється при вході у viewport:
```css
opacity: 0 → 1
transform: translateY(24px) → translateY(0)
transition: 0.6s ease
```
Бібліотека: AOS.js або Intersection Observer API нативно.

## 7. Адаптивність (Responsive)

| Breakpoint | Зміни |
|------------|-------|
| Desktop ≥ 1200px | Повна двоколонкова верстка, великі відступи |
| Tablet 768–1199px | Деякі блоки переходять в одну колонку, hero: текст та фото стек вертикально |
| Mobile ≤ 767px | Все в одну колонку, hamburger menu, карточки стаком, форма на повну ширину |

**Mobile-специфічні зміни:**
- Hero H1: 36–42px замість 56–72px
- Цифри-статистика: 3 колонки → 3 рядки або 2+1
- Послуги-список: залишити як є, тільки менший шрифт
- Форма: повна ширина

## 10. Що НЕ робити (антипатерни)

- ❌ Яскраві синій, зелений, помаранчевий кольори — руйнують серйозний тон
- ❌ Rounded corners > 4px на кнопках — виглядає несерйозно для юриста
- ❌ Стокові фото людей (не юриста) — знижують довіру
- ❌ Забагато анімацій і ефектів — відволікає від змісту
- ❌ Дрібний текст < 16px для основного тексту
- ❌ Форма з > 4 полями — знижує конверсію
- ❌ Відсутність мобільної версії
- ❌ Автопрогравання відео або музики
